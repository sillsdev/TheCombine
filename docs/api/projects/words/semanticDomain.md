# Semantic Domain

A semantic domain refers to a category of words. These words are nested by their number.

```
1. Universe, Creation
    1.1 Sky
        1.1.1 Sun
        1.1.2 Air
        1.1.3 Weather
    1.2 World
        1.2.1 Land
        1.2.2 Substance, Matter
        1.2.3 Solid, liquid, gas
```

## Raw Type

```typescript
{
    Name: String,
    Number: String,
}
```
